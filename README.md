SignUp Automation Test

Overview
Automates the complete Sign Up flow using Selenium WebDriver and TestNG, including form filling and document upload.

Tech Stack
- Java
- Selenium WebDriver
- TestNG
- ChromeDriver

How to Run
1. Open project in IntelliJ/Eclipse  
2. Ensure ChromeDriver matches Chrome version  
3. Update file path in:
   spg.uploadDocument("C://Users//absolute//IdeaProjects//project_vrit//QAFile.pdf");
4. Run `AllTests.java` as TestNG test  

Note
"Please enter your phone number for the signup in "Set up your Account", so that you can get OTP."
